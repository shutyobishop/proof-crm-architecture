Security Boundaries:
┌──────────────────────────────────────────────────────────────────────────┐
│  RLS: tenant_id = (auth.jwt()->>'tenant_id')::int    on 109/109 tables  │
│  Rate Limit: 5 POST /login attempts / min / IP                          │
│  FK Indexes: 216 indexes across 183 foreign key columns                 │
│  Monitoring: pg_stat_statements + ANALYZE every 6h                      │
│  Backup: pg_dump daily at 03:00 UTC, 7-day rolling retention            │
